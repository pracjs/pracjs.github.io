Datepicker Options

options
type date, month, year default is date
format YYYY-MM-DD type=date: default is YYYY-MM-DD
type=month: default is YYYY-MM
type=year: default is YYYY
class danger, primary, info, warn datepicker style class, default is danger
range false / true default is false
multiple false / true default is false
scope date scope user can select, default is from 1970-01-01 to 50 years later from now
btns Config buttons label and callback function
week Config for days of week
firstDay - 0: Sun, 1: Mon
month Config for month
year Config for year
