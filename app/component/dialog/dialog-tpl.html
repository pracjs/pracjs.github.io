Dialog Options

options
close true / false If show close button, default is true
but for Alert Dialog cannot be set and can only be false
footer true / false If show footer, default is true
but for info Info Dialog cannot be set and can only be false
size sm, lg, xl Modal dialog size, defaut is none. For detail please see Bootstrap Modal.
css {"min-width": "1000px"} Set the inline style for modal dialog
confirm JSON Detail see below
cancel JSON Not available for Alert Dialog. Detail see below
confirm
class primary Button css class, default is 'primary'.
but for Alert Dialog default is 'danger'.
For detail please refer to Buttons
label Confirm Button label, defaullt is 'Confirm'
waiting Waiting Waiting text on click confirm button, default is 'Waiting'
onConfirm function (notice) { notice(); } fired on click confirm button
cancel
class secondary Button css class, default is 'secondary'.
For detail please refer to Buttons
label Cancel Button label, defaullt is 'Cancel'
onCancel function () { } fired on click cancel button
disabled false Disable cancel button, default is false.
For Alert Dialog cannot be set and can only be true
